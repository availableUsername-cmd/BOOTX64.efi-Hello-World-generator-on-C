# BOOTX64.efi-Hello-World-generator-on-C
使用方法：将文件（efi/BOOTX64.efi）拖到U盘里的EFI/BOOT/BOOTX64.efi，进入电脑BIOS/UEFI界面，选择U盘启动
Use Method: Drag file(efi/BOOTX64.efi) into EFI/BOOT/BOOTX64.efi on U Disk, Enter BIOS Interface of the computer, choose U Disk option to launch, and you will see Hello World text on rightup concer of your computer
